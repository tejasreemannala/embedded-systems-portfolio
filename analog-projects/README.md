# Analog & Circuit-Based Projects

## Overview
This repository includes basic analog electronics projects developed to strengthen foundational understanding of signal amplification and sound generation.

---

## Projects Included
- Mini Audio Amplifier  
- Siren Generator Circuit  

---

## Description
These projects focus on component selection, circuit assembly, and understanding signal behavior in analog systems. Emphasis was placed on reducing noise, achieving stable output, and ensuring reliable circuit operation.

- Signal Amplification  
- Component Selection  
- Troubleshooting
